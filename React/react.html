• React : an open-source JavaScript library (V in MVC)
  • React DOM : an in-memory representation of the real DOM
  • State : managed between the component (like variables declared within a function); holds data and determines how the component renders and behaves
  • Phases of React Component LifeCycle : Initialization, Mounting, Updating (should, could, will Update), Unmounting (willUnmount)
  • Props : passed to the component (like function parameters)
  • Diffing : identifies what has changed in the virtual DOM
  • Reconciliation : updates the DOM with the results of diff
  • JSX : JavaScript + HTML syntax; JS expression in JSX through curly braces
  • Element : what you see on the screen; component : function/class which accepts input and return an element(via JSX)
  • Children : content between the opening and closing tag of JSX expression
