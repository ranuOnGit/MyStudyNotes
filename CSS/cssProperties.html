Specificity in decreasing order : ID, class, type (h1, p etc.), universal selector

Styling in decreasing order : inline, internal (style tag), external css file

Display : block, inline-block, flex

Horizontal Centering :
  • block level element - margin : auto
  • body - text-align : center
Visibility :
  • Display : none
    remove from flow, hide element, collapse the space
  • Opacity : 1 - visible to 0 - hidden
    hides element and preserves the space
  • Visibility : hidden
    hides element and preserves the space, used for transition or fade-in effect when becoming visible
Float : left, right (wraps around the element); also used for making columns by using height, width and float property

Clear : both, left, right (does not wrap around the element)

Position :
  • Static : default, poistioned according to the normal flow
  • Relative : positioned relative to its normal position (top, bottom, left, right)
  • Absolute : positioned relative to the parent with position relative
  • Fixed : positioned relative to viewport(screen), stays fixed as we scroll
  • Sticky : position toggles between relative and fixed, once the position is met in the viewport, then it sticks (needs one of the values - top, left, bottom, right)
