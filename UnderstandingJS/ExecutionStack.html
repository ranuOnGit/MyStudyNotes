everytime a function is called, a new execution context is created for that function; the 'this' variable is created for that function; the variables in it are set-up during the creation phase; every line of code is executed line-by-line
everytime a new function is invoked, a new execution context is created, put on top of the execution stack; when the funcion finishes, it is popped off; whatever's at the top is what's currently running, line-by-line, synchronously example 1:
function b() { } function a(){ b() } a() example 2:
function a(){ b() var c } function b(){ var d } a() var d
